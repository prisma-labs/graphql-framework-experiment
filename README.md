# Usage

<!-- usage -->

```sh-session
$ npm install -g pumpkins
$ pumpkins COMMAND
running command...
$ pumpkins (-v|--version|version)
pumpkins/0.0.0 darwin-x64 node-v10.17.0
$ pumpkins --help [COMMAND]
USAGE
  $ pumpkins COMMAND
...
```
